FIRST RUN SETUP (Automatic):

  The app will automatically:
  ✓ Check that Node.js and Python are installed
  ✓ Install Node.js dependencies (takes ~1-2 minutes)
  ✓ Create a Python virtual environment
  ✓ Install Python dependencies
  ✓ Start the app

  This only happens ONCE. Future launches are much faster (~5-10 seconds).

TROUBLESHOOTING:

  Q: "Node.js not found"
  A: Install Node.js from https://nodejs.org/ and restart PowerShell

  Q: "Python not found"
  A: Install Python from https://www.python.org/ and restart PowerShell
     (Check "Add Python to PATH" during installation)

  Q: "Port 8002 already in use"
  A: Another app is using that port. Restart your computer or:
     - Open Task Manager
     - Find and end the "python" process
     - Try launching again

  Q: The app window won't open
  A: 1. Make sure Node.js and Python are installed
     2. Try closing and re-running the launcher
     3. Check the terminal for error messages
